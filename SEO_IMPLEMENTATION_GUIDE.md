## 🎯 SEO Strategy Summary

### Core Principles:
1. **Keep Emotional H1s**: Don't change conversion-optimized headlines
2. **Add Keyword H2s**: Insert keyword-rich sections after hero
3. **Melbourne Geo-Targeting**: Add "Melbourne" + "Australian" throughout
4. **Semantic Keywords**: Include framework names (NIST, ISO 27001, Essential Eight, Privacy Act, TPB, DISP, PCI DSS)
5. **Link Internal Pages**: Link between related services and industries

### Key Phrases to Include:
- "Melbourne-based" / "Melbourne team"
- "Australian businesses" / "across Australia"
- "Victorian business landscape"
- Framework names: NIST, ISO 27001, Essential Eight
- Compliance: Privacy Act, APPs, TPB, DISP, PCI DSS
- Service terms: vCISO, cyber maturity, OT/IT, phishing simulation

### Internal Linking Strategy:
When adding new intro sections, link:
- "cyber maturity assessments" → /services/cyber-maturity
- "emergency incident response" → /panic-button
- "virtual CISO services" → /services/vciso
- "phishing training" → /services/cyber-awareness
- Industry mentions → /industries/[slug]
