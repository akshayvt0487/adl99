# Location OpenGraph Images

This directory contains OpenGraph (OG) images for location pages.

## Image Specifications
- **Size**: 1200x630px (Facebook/OpenGraph standard)
- **Format**: PNG or JPG
- **Design**: Location-specific branded images

## Design Template

Each OG image should include:
- ADL99 logo
- Location name prominently displayed
- Postcode
- Tagline: "Expert Cyber Security for [Location] Businesses"
- Brand colors: Primary blue (#0066CC) with white text
- Optional: Subtle Melbourne skyline or location landmark silhouette

## Fallback

If a location-specific image doesn't exist, the system will fall back to:
- `locations-default-og.png` (generic locations OG image)
- Or site-wide `og-image.jpg`
